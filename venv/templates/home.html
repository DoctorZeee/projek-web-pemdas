Tentang SolusiKerja

Website SolusiKerja adalah Website mencari lowongan kerja, Website ini ini hanya menampilkan informasi saja yang kami dapat dari internet. Di dalam website tidak terdapat transaksi ataupun pengupload an data.
Website ini menggunakan teknologi HTML, css dan framework flask Di dalam website ini kami menggunakan fitur fitur :
1. Halaman home :
Menampilkan penjelasan singkat mengenai kegunaan dan fungsi Website secara singkat.
2. Lowongan
Halaman ini berisi tentang informasi lowongan kerja yang tersedia, yang kami ambil dari beberapa website.
3. Kontak
