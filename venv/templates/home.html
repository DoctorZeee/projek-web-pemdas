Tentang SolusiKerja

Website SolusiKerja adalah Website mencari lowongan kerja. Website ini hanya menampilkan informasi saja yang kami dapat dari internet. Di dalam website tidak terdapat transaksi ataupun pengupload-an data.
Website ini menggunakan teknologi HTML, CSS, dan framework Flask. Fitur yang tersedia di dalam website ini:
1. Halaman Home:
Menampilkan penjelasan singkat mengenai kegunaan dan fungsi Website secara singkat.
2. Lowongan:
Halaman ini berisi tentang informasi lowongan kerja yang tersedia, yang kami ambil dari beberapa website.
3. Kontak:
Halaman untuk pengguna yang ingin menghubungi pengelola.
Teknologi yang digunakan:
1. Framework Python Flask
2. HTML
3. CSS
